COMO PUBLICAR A V97 SEM PERDER A BASE

IMPORTANTE:
- Não apague a pasta atual do repositório.
- Não apague static/data/dashboard-data.json.
- Não apague data/CONTROLE_DE_REQUISICOES_2026.xlsx.
- A V97 foi empacotada sem esses dois arquivos para preservar seus dados atuais.

PASSOS

1. Extraia o ZIP da V97.
2. Abra a pasta local do repositório dashboard-pcm-cloudflare.
3. Copie os arquivos extraídos para dentro do repositório.
4. Escolha “Substituir os arquivos no destino”.
5. Confirme que static/data/dashboard-data.json continua no repositório.
6. Abra o GitHub Desktop.
7. Use o resumo: V97 confiabilidade e navegação.
8. Clique em Commit to main.
9. Clique em Push origin.
10. Espere o deploy da Cloudflare concluir.
11. Abra o dashboard e pressione Ctrl + F5 uma vez.

PARA ATUALIZAR A PLANILHA DEPOIS

1. Substitua data/CONTROLE_DE_REQUISICOES_2026.xlsx.
2. Feche o Excel.
3. Execute ATUALIZAR_DADOS.cmd.
4. Confira as contagens exibidas.
5. Verifique se foram alterados:
   - static/data/dashboard-data.json
   - static/data/version.json
   - data/quality-report.json
6. Faça Commit e Push.
