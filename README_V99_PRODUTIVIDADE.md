# Dashboard PCM V99.3 — Produtividade Operacional

## Recursos

- busca múltipla;
- seleção de linhas;
- copiar resumo;
- colunas configuráveis;
- filtros salvos;
- URL compartilhável;
- gaveta de detalhes;
- exportação Excel real.

## Publicação

1. Substitua os arquivos da versão anterior.
2. Coloque a planilha oficial na pasta `data`.
3. Execute `ATUALIZAR_DADOS.cmd`.
4. Confirme `VALIDAÇÃO V99.3: OK`.
5. Faça Commit e Push.
