# Guia rápido

## Visão Geral

Clique em qualquer KPI, etapa, fornecedor, solicitante, equipamento ou item de atenção para abrir o Acompanhamento já filtrado.

## Acompanhamento

- Digite ORC, OS, pedido, NF, prefixo, fornecedor ou equipamento.
- Use os atalhos Sem lançamento, Sem pedido e Sem NF.
- Clique na linha para abrir o Drawer.

## Drawer

O topo mostra imediatamente **O que está faltando**. As abas organizam Resumo, Rastreabilidade, Documentos e Histórico.

## Importações

Use sempre a tela Importações para atualizar a base oficial compartilhada. Não altere arquivos do projeto para atualizar dados.

## TV

A rota `/tv` mostra KPIs, fluxo, atenção, fornecedores, solicitantes e equipamentos em modo de leitura.
