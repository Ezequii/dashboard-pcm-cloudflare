COMO ATUALIZAR A PLANILHA - DASHBOARD PCM CLOUDFLARE PAGES

Essa versão é estática. O dashboard não lê Excel diretamente na internet.
Ele lê este arquivo:

  static/data/dashboard-data.json

Para atualizar:

1. Coloque a nova planilha nesta pasta:

  data/CONTROLE_DE_REQUISICOES_2026.xlsx

2. Dê dois cliques em:

  ATUALIZAR_DADOS.bat

3. Espere aparecer a mensagem de sucesso.

4. Abra o GitHub Desktop.

5. Faça commit:

  Atualizar base do dashboard

6. Clique em Push origin.

7. A Cloudflare Pages vai publicar sozinha em alguns minutos.

IMPORTANTE:
- O arquivo Excel dentro da pasta data/ não deve ser enviado ao GitHub.
- O .gitignore desta versão já ignora data/*.xlsx.
- O arquivo publicado é apenas static/data/dashboard-data.json.
